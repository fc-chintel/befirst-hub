# BE:FIRST HUB 💛

BE:FIRSTの **新着ニュース・最新動画・今後の予定・公式リンク** をひとつの画面にまとめた、
iPhoneのホーム画面に追加して使える非公式ファンアプリです(個人利用向け)。

---

## このフォルダの中身

| ファイル / フォルダ | 役割 |
|---|---|
| `index.html` | アプリの画面本体 |
| `assets/` | デザイン(style.css)と動き(app.js) |
| `data/` | 表示するデータ。news.json / videos.json は自動生成、**events.json は手で編集OK** |
| `scripts/` | データ取得プログラムとアイコン生成プログラム |
| `icons/` | アプリのアイコン画像 |
| `プレビュー起動.bat` | **ダブルクリックでパソコン上でアプリを見られます** |
| `データ更新.bat` | **ダブルクリックで最新ニュース・動画を取得します** |
| `.github/workflows/` | GitHub公開後に1日4回自動更新してくれる設定 |

## パソコンで見てみる

1. `データ更新.bat` をダブルクリック(最新データを取得)
2. `プレビュー起動.bat` をダブルクリック → ブラウザで開きます
3. やめるときは黒い画面(コマンドプロンプト)を閉じるだけ

## スケジュールを直したいとき

`data/events.json` をメモ帳などで開いて編集してください。
1行が1つの予定です。日付が未定のものは `"approx": true` を付けると「日付未定」と表示されます。

## iPhoneで使えるようにする(インターネット公開)

奥さんのiPhoneで使うには、アプリをインターネット上に置く必要があります。
無料の「GitHub Pages」を使います。手順はざっくり:

1. [GitHub](https://github.com/) で無料アカウントを作る
2. 新しいリポジトリ(置き場所)を作り、このフォルダの中身をアップロード
3. リポジトリの Settings → Pages → Branch を `main` にして保存
4. 数分後に `https://あなたの名前.github.io/リポジトリ名/` でアプリが見られます
5. **アップロードした瞬間から、1日4回の自動データ更新も勝手に動き始めます**

※ この手順は Claude Code に「GitHubに公開するのを手伝って」と頼めば一緒に進められます。

## iPhoneのホーム画面に追加する(奥さんのスマホで)

1. Safariで上記のURLを開く
2. 共有ボタン(四角から矢印が出ているアイコン)をタップ
3. 「ホーム画面に追加」をタップ

これでアイコンから普通のアプリのように起動でき、オフラインでも直近のデータが見られます。

---

### データの出どころと注意

- ニュース: 公式サイトが一般公開しているRSSフィード(befirst.tokyo/feed/)
- 動画: YouTubeが公式提供しているチャンネルRSSフィード
- どちらも「新着のお知らせを受け取るための公開された仕組み」で、1日4回の取得なのでサイトに負荷はかけません
- 個人・家庭内での利用を想定しています。画像や本文の転載・再配布はしないでください
